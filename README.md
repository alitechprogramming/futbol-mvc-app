# Aplicación Web sobre noticias de fútbol, jugadores y clubes.
Aplicación web para el curso de patron de arquitectura MVC con .NET Core MVC 7

## Cómo configurar el proyecto para ejecutarlo forma local:

**1** - Clonar el repositorio con el comando **git clone https://github.com/alitechprogramming/futbol-mvc-app.git** en la carpeta que deseen.

**2** - Dirigirse a la carpeta donde fue clonado el proyecto y hacer doble clic en **AplicacionMVC.csproj**, esto les abrirá el proyecto con el IDE Visual Studio 2022, este es el IDE que estamos utilizando para el proyecto, tambien pueden utilizar Visual Studio Code si desean.

**3** - El proyecto esta formado por dos (2) ramas la rama **main** y la rama **develop** los cambios los iremos realizando en la rama develop así que pueden abrir una linea de comandos en la carpeta del proyecto y escribir el comando **git checkout develop**.

**4** - Es importante que tengan instalado Git en su maquina, de lo contrario estos comandos no les funcionarán.
